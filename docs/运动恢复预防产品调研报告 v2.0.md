## 1. 执行摘要

### 1.1 定位调整说明

**原定位：** 运动康复平台（专业康复、术后恢复）  
**新定位：** 运动恢复预防指南（健身爱好者的恢复助手）

**调整原因：**
- 创始人无专业康复背景，做专业康复风险高
- 健身爱好者视角更真实、接地气
- 避开医疗资质风险，专注"预防"和"恢复"

### 1.2 新愿景

打造**健身爱好者首选的运动恢复平台**，让每个人都能科学恢复、远离损伤。

### 1.3 核心价值

| 价值点 | 描述 |
|--------|------|
| 🎯 **精准场景** | 专注健身/运动后的恢复需求 |
| 📹 **跟练视频** | 简单易懂的拉伸/放松教程 |
| 📊 **恢复追踪** | 记录恢复状态，看到进步 |
| ⚠️ **风险边界** | 明确声明：严重伤情建议就医 |

## 2. 产品定位调整

### 2.1 产品边界（非常重要！）

| ✅ 可以做 | ❌ 不做 |
|---------|--------|
| 运动后拉伸放松 | 疾病治疗 |
| 肌肉疲劳恢复 | 严重损伤诊断 |
| 小伤小痛自我处理 | 术后康复 |
| 损伤预防指导 | 处方级康复方案 |
| 恢复知识科普 | 替代医生建议 |

### 2.2 产品 Slogan

> "练得好，更要恢复好"  
> "你的私人恢复教练"

### 2.3 品牌调性

- **专业但不高冷** — 用健身爱好者能懂的语言
- **实用主义** — 每个动作都有明确目的
- **安全第一** — 反复强调风险边界
- **陪伴感** — 像身边的健身老友

## 3. 市场规模与趋势

### 3.1 目标市场规模

| 指标 | 数据 | 来源 |
|------|------|------|
| 中国健身人群（2025） | **3.5 亿+** | 三体云动《中国健身行业报告》 |

| 运动恢复/防护市场 | **约 500 亿元** | 行业估算 |

### 3.2 目标用户占比

```
健身人群 3.5 亿
├── 有恢复意识（主动拉伸/放松）: ~30% = 1 亿
├── 遇到过运动损伤：~50% = 1.75 亿
└── 愿意为恢复付费：~10% = 3500 万

可触达市场：3500 万付费用户 × ¥30/月 = ¥10.5 亿/年
```

### 3.3 市场趋势

| 趋势 | 说明 | 机会 |
|------|------|------|
| **健身人群增长** | 健康意识提升 | 用户基数扩大 |

| **内容付费** | 用户愿为专业付费 | 商业模式成熟 |

## 5. 用户画像与痛点

### 5.1 核心用户群体

#### 用户 A：健身房爱好者

```
年龄：25-35 岁
运动频率：每周 3-5 次
特征：力量训练为主，追求增肌

痛点：
- 练完肌肉酸痛，影响下次训练
- 不知道练后怎么拉伸
- 担心动作不标准受伤

需求：
- 针对不同肌群的拉伸教程
- 快速缓解酸痛的方法
- 动作标准性指导

付费意愿：中（¥20-50/月）
```

#### 用户 B：跑步爱好者

```
年龄：25-45 岁
运动频率：每周 3-4 次
特征：跑步 5-10 公里，关注配速

痛点：
- 膝盖、脚踝偶尔不舒服
- 跑后不知道怎么放松
- 担心长期跑步伤膝盖

需求：
- 跑者专属拉伸方案
- 关节保护训练
- 恢复进度追踪

付费意愿：中（¥20-50/月）
```

#### 用户 C：居家健身人群

```
年龄：20-40 岁
运动频率：每周 2-3 次
特征：跟练视频，设备有限

痛点：
- 练完浑身疼，不知道正常吗
- 想拉伸但不知道动作
- 担心动作做错受伤

需求：
- 简单易懂的恢复教程
- 无需器械的恢复动作
- 恢复知识科普

付费意愿：低 - 中（¥10-30/月）
```

#### 用户 D：运动损伤恢复期人群

```
年龄：20-50 岁
特征：轻微扭伤/拉伤恢复期

痛点：
- 不知道什么时候能恢复训练
- 想活动但怕二次受伤
- 医院排队太麻烦

需求：
- 恢复期训练指导
- 风险边界提醒
- 必要时就医建议

付费意愿：高（¥50-100/月）
```

### 5.2 用户场景地图

```
运动前 → 运动中 → 运动后 → 恢复期 → 下次运动
  │        │        │        │         │
  │        │        └─→ 拉伸放松（核心场景）
  │        │        └─→ 筋膜放松
  │        │        └─→ 营养补充建议
  │        │
  │        └─→ 运动中不适 → 处理建议
  │
  └─→ 动态热身 → 预防损伤
```

### 5.3 用户需求优先级

| 需求 | 优先级 | 现有产品满足度 | 机会 |
|------|-------|--------------|------|
| 练后拉伸教程 | P0 | ⭐⭐⭐ | 做得更系统 |
| 缓解酸痛方法 | P0 | ⭐⭐ | 深度优化 |

| 风险边界提醒 | P2 | ⭐ | 建立信任 |

## 6. 差异化策略

### 6.1 核心价值主张

**"不是康复，是恢复；不是治疗，是预防"**

### 6.2 差异化要点

| 维度 | 竞品 | 本项目 |
|------|------|--------|
| **定位** | 健身平台附带恢复 | 专注恢复预防 |
| **内容** | 通用拉伸 | 分场景/分肌群 |
| **追踪** | 无 | 恢复效果量化 |
| **风险** | 模糊 | 清晰边界 |
| **视角** | 教练视角 | 健身爱好者视角 |

### 6.3 内容差异化

**Keep 的内容组织：**
```
健身 → 瑜伽 → 拉伸
├── 全身拉伸
├── 腿部拉伸
└── 肩部拉伸
```

**本项目的组织：**
```
场景 → 部位 → 问题
├── 练后恢复
│   ├── 练腿日 → 腿部拉伸 + 筋膜放松
│   ├── 练胸日 → 胸部拉伸 + 肩关节放松
│   └── 练背日 → 背部拉伸 + 脊柱放松
├── 酸痛缓解
│   ├── DOMS 缓解
│   ├── 肌肉僵硬
│   └── 关节不适
└── 损伤预防
    ├── 膝盖保护
    ├── 肩膀保护
    └── 腰部保护
```

### 6.4 功能差异化

| 功能 | 说明 | 竞品有无 |
|------|------|---------|
| 恢复日志 | 记录每次恢复训练 | ❌ |
| 酸痛评分 | 追踪酸痛变化 | ❌ |
| 恢复计划 | 基于训练自动生成 | ❌ |
| 风险自测 | 判断是否需要就医 | ❌ |
| 动作纠正 | AI 识别动作标准度 | ⭐（Keep 部分） |

## 7. MVP 功能规划

### 7.1 功能架构

```
运动恢复指南
├── 用户系统
│   ├── 注册/登录
│   └── 运动档案（常练项目、伤病史）
├── 恢复库
│   ├── 按场景（练后/酸痛/预防）
│   ├── 按部位（腿/背/胸/肩/核心）
│   └── 按器械（无器械/泡沫轴/筋膜枪）
├── 跟练系统
│   ├── 视频播放
│   ├── 动作讲解
│   └── 计时提醒
├── 恢复日志
│   ├── 训练记录
│   ├── 酸痛评分
│   └── 恢复进度
├── 智能推荐
│   ├── 基于训练推荐
│   └── 基于状态推荐
└── 知识科普
    ├── 恢复知识
    └── 风险边界
```

### 7.2 MVP 功能（第一版）

| 功能 | 描述 | 优先级 | 预估工作量 |
|------|------|--------|-----------|
| 用户注册 | 基础账号系统 | P0 | 2 天 |
| 运动档案 | 记录常练项目、部位 | P0 | 2 天 |
| 恢复库（20 个动作） | 分场景/部位的恢复视频 | P0 | 5 天 |
| 跟练播放器 | 视频 + 计时 + 讲解 | P0 | 3 天 |
| 恢复日志 | 记录训练和感受 | P1 | 2 天 |

| 知识科普（10 篇） | 恢复知识文章 | P1 | 3 天 |

**MVP 总工作量：** 约 18 天（1 人）

## 8. 风险评估与应对

### 8.1 核心风险

| 风险 | 概率 | 影响 | 应对方案 |
|------|------|------|---------|
| 用户受伤追责 | 中 | 高 | 明确免责声明 + 风险边界 + 保险 |
| 被认定非法行医 | 中 | 高 | 定位"恢复预防"，不做"治疗康复" |
| 内容专业性质疑 | 高 | 中 | 找专业顾问审核 + 持续学习 |
| 用户付费意愿低 | 中 | 中 | 免费 + 增值，证明效果 |

### 8.2 免责声明建议

```
⚠️ 重要提示

本产品提供的内容仅供健身爱好者参考，用于运动后恢复和损伤预防，
不替代专业医疗建议、诊断或治疗。

以下情况请及时就医：
- 剧烈疼痛或疼痛持续超过 1 周
- 关节活动明显受限
- 疑似骨折、韧带撕裂等严重损伤
- 术后康复需求

使用本产品即表示您同意：
1. 根据自身情况谨慎练习
2. 如有不适立即停止
3. 理解本产品不替代专业医疗
```

### 8.3 内容审核机制

- **第一阶段：** 创始人自学 + 公开资料整理
- **第二阶段：** 邀请康复师/健身教练做顾问（付费审核）
- **第三阶段：** 建立内容审核流程
