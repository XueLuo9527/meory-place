# 🎨 记忆宫殿 UI 交互 Demo

> **创建时间：** 2026-03-09

## 📋 Demo 列表

### 🏰 方案 A：3D 宫殿漫游
**文件：** `demo-a-3d-roam.html`

**体验特点：**
- 第一人称视角（FPS）
- WASD 移动 + 鼠标转向
- 3D 空间探索
- 记忆物体悬浮展示

**操作说明：**
1. 点击"进入宫殿"开始
2. `W/A/S/D` 移动
3. `鼠标` 转向
4. 靠近记忆物体后按 `E` 查看

**技术栈：** Three.js

**推荐场景：** 桌面端，追求极致沉浸感

---

### 🎨 方案 B：2.5D 等轴测视图
**文件：** `demo-b-isometric.html`

**体验特点：**
- 等轴测俯视视角
- 点击房间进入
- 拖动旋转视角
- 滚轮缩放

**操作说明：**
1. 点击房间高亮
2. 点击记忆卡片（🍎🍌等）查看
3. 拖动空白处旋转视角
4. 鼠标滚轮缩放

**技术栈：** CSS 3D Transform + 原生 JS

**推荐场景：** 全平台默认方案 ⭐

---

### 🌌 方案 C：时间线穿梭
**文件：** `demo-c-timeline.html`

**体验特点：**
- 星河粒子可视化
- 时间线穿梭
- 标签筛选
- 搜索功能

**操作说明：**
1. 拖动底部时间线滑块
2. 点击左侧标签筛选
3. 右上角搜索框搜索
4. 点击星星（记忆）查看

**技术栈：** Canvas + 粒子系统

**推荐场景：** 大量记忆，时间维度浏览

## 🚀 如何运行

### 方法 1：直接打开（推荐）
```bash
# 在浏览器中直接打开 HTML 文件
open demos/demo-a-3d-roam.html
open demos/demo-b-isometric.html
open demos/demo-c-timeline.html
```

### 方法 2：本地服务器
```bash
# 进入 demos 目录
cd demos

# Python 3
python -m http.server 8080

# 或 Node.js
npx serve

# 访问 http://localhost:8080
```

## 💡 使用建议

### 选择方案 A 如果：
- ✅ 目标用户主要是桌面端
- ✅ 追求极致视觉体验
- ✅ 有性能优化预算
- ✅ 愿意投入开发时间

### 选择方案 B 如果：
- ✅ 需要全平台支持
- ✅ 快速上线 MVP
- ✅ 平衡体验和成本 ⭐ **推荐**
- ✅ 喜欢独特视觉风格

### 选择方案 C 如果：
- ✅ 记忆数量巨大（1000+）
- ✅ 时间维度很重要
- ✅ 用户需要快速浏览
- ✅ 喜欢抽象艺术风格
